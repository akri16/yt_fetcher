## Youtube Video Fetcher

### Services
  - DB
  - Poller
  - API

## TeckStack
  - FastAPI
  - MongoDB
  - Docker, Docker Compose
  - Youtube Data V3 API

### Steps To Run
  - Use the docker compose file and provide the YT_API_KEY env variable
    ```cmd
      export YT_API_KEY=<API_KEY>
      docker compose up
    ```
